# Carelog 요금·등급 정책 (초안 v2)

> **상태**: 핵심 구조 확정(2026-06-19, 대표님). 일부 숫자(포함 분량·토큰 단가)는 권장값 — 조정 가능.
> **소유**: 다온(PM). 이 문서가 SSOT. 음성 보존·재청취 구현은 `specs/009-audio-archive`가 등급 게이트를 참조.

## 0. 핵심 원칙
1. **3주체 가치 사다리**: 직원(편함) → 병원(운영: 유실방지·시간절감·환자경험) → 병원(경영: 분석·마케팅). 등급이 윗 주체의 가치를 하나씩 더 얹는다.
2. **정액 = "기능·보관" 차등 / 토큰 = "전사 분량(변동원가)" 분리.** 전사는 쓸수록 돈 나가므로 정액에 묶지 않고 사용량으로 회수.
3. **상담보드(코어)는 팔지 않는다.** 무료 맛보기의 "와우"를 만드는 심장이라 전 등급 기본. 모듈로 파는 건 *주변부 고급기능*과 *보드의 프리미엄 옵션*뿐.
4. **유료 진입은 쉽게.** 출시는 단순한 가격표(Free + Standard), Pro·모듈·토큰은 기능 완성·수요 확인 후.
5. **텍스트 차트(의무기록)는 등급 무관 영구.** 등급 레버는 음성·부가기능에만.

## 1. 과금 구조
- **정액(월정액, 병원 단위)** — 제품 등급 + 월 포함 토큰(분).
- **토큰(사용량)** — 전사 **1분 = 토큰 1**. 매월 포함분 리셋, 초과 시 충전.
- **모듈(부가 구매)** — 주변부 고급기능(향후).
- **Enterprise(문의)** — 다지점·맞춤은 별도 협의 상품으로 분리.
- 멤버 수에 따른 등급 차등 **없음**.

## 2. 등급 컨셉
- 🟢 **Free — "한 명의 와우"** : 한 직원이 "이거 진짜 편하네"를 느끼는 체험판. 맛보기·후킹.
- 🔵 **Standard — "병원의 상담~차팅을 실무로"** : 매일의 상담·차팅을 Carelog로 돌리는 기본판. 유실방지·시간절감·환자전달.
- 🟣 **Pro — "상담 데이터를 경영·마케팅 자산으로"** : 분석·CRM·다량 사용·긴 보관. 경영 가치(상위 비전 입구).
- ⚫ **Enterprise — "다지점·맞춤"** : 문의형. 다지점·전용 연동·맞춤 보관.

## 4. 토큰(전사 분량) 모델
- **1 토큰 = 전사 1분.** 녹음→AI 변환 시 사용한 분만큼 토큰 차감.
- 월 포함 토큰은 **매월 리셋**(이월 없음 가정). 소진 시 **충전(팩 단위)**.
- 원가 ≈ **₩10/분**(Whisper STT ~₩8 + Claude 요약 ~₩2). 충전 판매가 **₩30~50/분(초안, 원가 3~5배)** — 예: 100분 팩 ₩5,000.
- SMS 발송 등 다른 실비 항목도 향후 동일 토큰/사용량 과금으로 흡수 가능.
- ⚙️ 실제 충전·결제(PG)·잔액 관리 = 빌링 시스템(향후 별도). 지금은 정책만.

## 5. 모듈(부가 구매, 향후)
- **팔 수 있는 것(주변부 고가치)**: 상담 분석·CRM, 라미네이트 등 상담 시뮬/자료, 특정 EMR 연동, 대량 문자 캠페인, 보드 프리미엄 옵션(실시간 받아쓰기·화자분리·AI 상담코칭).
- **팔지 않는 것(코어)**: 상담보드 기본, 녹음→AI 기록→복붙, 다기기 알림. → 무료 와우의 원천이라 비매.

## 6. Free 전략 — 맛보기 + 넛징
- 월 30분 + 음성 최근 3개 + 분석 맛보기로 "와 좋다" 경험.
- 넛징 트리거(UI): 사용량 80% 도달 / 4번째 음성 저장 시 정리 안내 / 분석 화면 샘플+"Standard에서 전체".

## 7. 내부 / 파트너
- 예미안·케어로그치과·파일럿(도은쌤) = **Pro(또는 Enterprise) 무상**.

## 8. 미결 / 다음
- 포함 분량(30/300/1,000)·토큰 충전 단가 최종 확정.
- 토큰 충전·결제(PG) 빌링 시스템 = 향후 별도 스펙.
- 모듈 카탈로그·가격.
- ⚖️ 의료법상 음성 보존의무·동의 형식 최종 판단(운영 확인). 보존기간 차등은 "음성=부가 백업, 텍스트=의무기록" 전제.

## 9. 구현 연결
- 기관(`institutions`)에 **`plan`(free | standard | pro | enterprise)** 필드 → 기능 게이트 단일 출처. 기본 free, 내부/파트너 pro/enterprise.
- 음성 보존기간·재청취 = `specs/009-audio-archive`가 `plan` 참조(Free 최근3 / Standard 90일 / Pro·Enterprise 1년+).
- 토큰(전사 분량) 미터·충전 = 별도(향후). 본 문서는 정책.
