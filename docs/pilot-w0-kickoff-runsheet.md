## 2. 분 단위 진행

### 0–5분 · 라포 + 프레이밍 🤝
- "와줘서 고마워. 오늘은 시험 보는 자리가 전혀 아니야."
- **핵심 멘트**: "케어로그가 못 쓰겠으면 그건 **네 잘못이 아니라 제품 잘못**이야. 그 '못 쓰겠는 순간'을 찾아주는 게 이 4주 동안 네가 할 일이야 — 그게 제일 값진 데이터야."
- 보상·기간 가볍게: "4주, 주 1시간 대화, 세전 12만 원(주 3만 상당). 일을 더 시키는 게 아니라 **솔직한 피드백에 대한 사례**야."

### 5–10분 · 왜 너냐 (존중) 🙌
- "1년차가 사실 상담을 직접 주도할 일은 적지만, **기록을 가장 많이 하는 사람**이잖아. 그래서 마찰을 제일 많이 느껴."
- "신입인 네가 빠짐없이 기록해낼 수 있으면, 그건 케어로그가 진짜 쓸만하다는 **가장 강한 증거**가 돼. 그래서 네가 적임자야."

### 10–25분 · 케어로그 한 바퀴 (다온이 시연) 🎬
> 말로 설명하지 말고 **흐름을 보여준다**. 이 한 줄기에 핵심이 다 들어 있다.

1. **홈** — "여기서 시작 버튼 하나가 전부야."
2. **"상담 기록 시작" 1탭 → 즉시 녹음** — 체어·환자 안 골라도 바로 녹음됨을 강조 ("급할 때 빠짐없이")
3. **AI 전사** — 녹음이 글로 자동 변환됨
4. **상담보드** — 본문·그림·체어·참여자·처방을 한 화면에서 채움
5. **환자 연결** — 나중에 환자에게 붙이기 (지금 안 골라도 됨)
6. **환자 포털** — 환자가 SMS로 받아 **자기 상담 기록을 평생 보관**하는 화면까지 보여주기
- **마무리 한마디**: "기능 많아 보여도 네가 외울 건 딱 하나, **1탭 녹음**. 나머지는 천천히."

### 25–40분 · 도은이 손으로 1건 + 멀티기기 세팅 📱
> 보는 것과 하는 것은 다르다. **직접** 끝까지 한 번. (워크스트림 A 동반 완료)
> §4 체크리스트 — 도은이 주력 기기에서 1회:

- ☐ 브라우저 Chromium 계열 확인 (체어 PC/노트북 = Chrome/Edge)
- ☐ 개인 계정 **로그인 유지**
- ☐ **PWA 설치**(홈 화면 추가) — 마이크 권한 1회만 허용하면 재요청 없음
- ☐ 마이크 권한 **허용** + **테스트 녹음 1회**
- ☐ **알림 소리 켜기**(🔔 1회) — 다화면 실시간 알림용
- ☐ (선택) Web Push 구독 허용
- ☐ **기기 = 체어 매핑**: 그 기기에서 해당 체어로 한 번 기록 → 이후 "{체어명} 바로 녹음" 1탭
- ☐ **화면 자동잠금**(예: 5분) — 공용 화면 환자정보 노출 방지(헌법 I)
- ☐ 태블릿(iOS) 쓰면: 소리·푸시 제약 있음을 미리 안내(보조 기기로)

### 40–50분 · 업무 관찰 / 통증 특정 🔍
- **질문**: "하루 일하면서 상담 내용을 **기록 못 하고 넘어가거나**, 기록이 **제일 귀찮은 순간**이 언제야?"
- 구체화: 그게 **어떤 체어/어떤 상황**인지 (PII 없이 상황만)
- **Top 1~2 특정** → 즉석으로 W0 피드백 카드에 적기
- **범위 합의**: 케어로그가 풀 문제인지 솔직하게. 범위 밖이면 "그건 우리가 풀 게 아니야"라고 빼기

### 50–58분 · 톤·일정·가드레일 📅
- 주간 회의 구조 예고: **15분 개선 확인 / 30분 옆에서 관찰 / 15분 다음 우선순위 1개**
- **과부하 가드레일 재확인**: "설문지·사용일지 없어. 평소처럼 쓰면 돼. 분석은 우리가 짊어져."
- "하루 끝에 '오늘 편했음/짜증' **한 줄**은 받으면 좋지만 **강제 아님**."
- **다음 회의 시간 픽스** (1주차)

### 58–60분 · 마무리 ✅
- 1주차 목표 **딱 하나만**: "이번 주는 그냥 1탭 녹음을 평소 상담에서 써보는 것."
- **클로징 멘트**: "쓰다가 막히거나 짜증나면, 참지 말고 **그 자리에서 바로** 말해줘. 그게 핵심이야."

## 3. 진행 중 다온 주의사항

- **데모를 건너뛰고 관찰부터 가지 말 것.** 도은이는 0주라 케어로그를 모른다 → 데모(10–25분)·직접 1건(25–40분)을 먼저 깔아야 통증 대화가 굴러간다.
- 도은이가 머뭇거리는 지점 = **금광**. 멘트로 넘기지 말고 **탭 수·머뭇거린 위치**를 메모.
- **PII 금지**: 메모·카드에 환자 식별정보 적지 않기. 체어명·상황만(§8).
- 부담 신호(피곤·위축) 보이면 즉시 줄인다. "오늘 여기까지 해도 충분해."
