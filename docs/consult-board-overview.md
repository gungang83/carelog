# Carelog 상담보드, 지금 어떻게 생겼나 (쉬운 설명)

> 공유용 개요(비개발자 대상). 기술 상세는 `docs/architecture.md`, 스펙별 문서는 `specs/`.
> 작성: 다온 · 2026-07-08 (세션 66 기준 — spec 008·010·016·020·021·025·026·027·028 반영)

## 한 문장으로

**"버튼 하나 누르면 녹음이 시작되고, 상담이 흘러가는 동안 보드 위에서 자료를 보여주고·그리고·견적까지 만들면, 끝났을 때 정리된 상담 기록이 되어 있는 작업대"**

## 시작 — 준비 없이 바로

홈에서 **[상담 기록 시작]**을 누르는 순간 녹음이 켜지고 보드가 열린다. 체어·참여자는 미리 고르지 않아도 된다 — 상담하면서, 혹은 끝나고 보드 안에서 채운다. (체어는 이 컴퓨터에서 마지막에 쓴 체어가 자동 선택)

## 상담 중 — 보드는 작업대

- **말은 그냥 하면 된다.** 녹음이 담고, AI가 `[진찰 소견]` `[처방·권고]` 섹션의 평문으로 정리한다(차트에 붙여도 깨지지 않게 마크다운 없음). 중간 **일시정지/이어서 녹음** 가능.
- **📚 자료** — 미리 등록한 설명 이미지(임플란트 단계·치식도·동의서…)를 꺼내 **전체화면으로 크게 열고 펜/마우스로 그려가며** 설명 → "기록에 담기"로 그린 그대로 기록에. 백지·모눈·줄노트 즉석 그림, 설명 영상 링크(환자 전달용)도.
- **₩ 견적** — 치료 항목 탭탭 → 수량·금액 조정 → 합계 계산된 [치료비 견적] 블록 삽입.
- **글·사진 편집** — 타이핑·사진 붙여넣기·사진 주석·이미지 글감싸기 배치·⛶ 전체화면.
- 에디터 위 안내: "이 내용은 [체어명] 상담 기록으로 저장됩니다".

## 보드를 닫아도 상담은 계속

X를 누르거나 다른 화면으로 가도 **녹음은 끊기지 않는다.** 화면 구석에 `🔴 상담중 (녹음중) 12:34` 표시가 상주하고, 누르면 복귀. 데스크톱 크롬은 "항상 위" 미니창으로 **차트 프로그램 위에도** 표시 가능.

**방치 안전망**: 조작도 대화 소리도 없이 10분(설정 가능) → "계속하시겠어요?" 경고 → 5분(설정 가능) 무응답 → **자동 종료·저장**(폐기 아님 — 이어서 수정 가능. 체어 미선택이면 '미지정'으로 저장). 기준은 설정 → 상담 안전망.

## 끝 — 저장은 서버가 마무리

**[상담 종료 및 저장]** → 음성만 서버로 올라가고 화면은 즉시 자유(탭 닫아도 OK). 서버가 전사·요약 후 알림. 기록 = 정리된 요약 + 그린 그림 + 견적. 이후:
- **전체 복사** → 덴트웹 등 차트에 평문 붙여넣기
- **환자 연결** → 환자 포털 전달(환자 보관·열람)
- **확인 꼬리표** → 차트 이관 전 확인 항목 표시

## 세 가지 상담 환경, 화면은 하나

① 체어 정면 모니터+마우스 ② 체어 태블릿+펜 ③ 상담실 컴퓨터+키보드 — 같은 보드 하나로 대응(큰 버튼·전체화면=마우스·터치, 에디터 중심=상담실).
